# NOVA AI

> AI-Powered Student Workspace

NOVA AI is an intelligent student productivity platform that helps learners summarize notes, generate quizzes, create study plans, and organize learning resources using Agentic AI.

## Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion

### Backend
- FastAPI
- Python

### AI
- Gemini API
- LangGraph
- LangChain
